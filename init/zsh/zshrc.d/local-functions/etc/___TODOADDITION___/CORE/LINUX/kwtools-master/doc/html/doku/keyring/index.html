Schlüsselring

Das Programm apt-key wird benutzt, um eine Liste von Schlüsseln zu verwalten, die APT benutzt, um Pakete zu authentifizieren. Pakete, die durch Benutzung dieser Schlüssel authentifiziert wurden, werden als vertrauenswürdig betrachtet.

Notiz: Eine automatische Installation von Software-Paketen funktioniert nur wenn der dementsprechende Schlüssel installiert ist.

Der debian-kwtools-keyring Schlüssel authentifiziert alle Pakete dieser Seite.
