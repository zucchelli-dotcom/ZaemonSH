1.5.1 kwauth (testing)

Mit diesem Script kann man die Methode zur Authentifizierung des Systems einstellen. Zur Zeit werden bei Debian nur die Authentifizierungs-Methoden PAM, LDAP und NIS unterstützt, falls installiert.

Notiz: Benoetigt root Rechte. Bei OpenSuSE wird das Programm yast2 -> Netzwerk Services, bei CentOS/Fedora authconfig-tui benutzt.
