kwrecord

Dieses Script brennt Daten, Iso Images, Audio, Video-CD/DVD's und kopiert CD/DVD's auf CD-+R(W)/DVD-+R(W)/RAM. Die zugrundeliegende Funktion cd_dvd_burn kann man in jedem Script nutzen. Beispiel: cd_dvd_burn --isocd, brennt ein oder mehrere ISO Images auf CD. Schauen Sie dazu in den Header der Funktion. Die Funktion cd_dvd_burn prüft ob ein leerer Rohling eingelegt wurde oder ob er erst gelöscht/formatiert werden muss.

Notiz: Dieses Script benutzt die Programme cdrdao und growisofs. Multisession CD/DVD's wird noch nicht unterstützt.

1 Konfiguration

Hier kann man ein paar globale Brenn(er) Einstellungen tätigen.

CD-Brennprogramm: Es wird geprüft ob cdrdao installiert ist und die richtigen Rechte hat. Die Rechte werden seit cdrdao Version 1.2.2 nicht mehr auf setuid geprüft, da man seitdem auch mit normalen Rechten brennen kann.

DVD-Brennprogramm: Es wird geprüft ob growisofs installiert ist und die richtigen Rechte hat.

CD-Grösse: Hier stellt man die Grösse des Rohlings ein. Zur Auswahl stehen 650, 680, 700 und 800Mb.

Speed: Hier stellen Sie die Brenngeschwindigkeit für CD ein. Meistens steht auf der CD-R(W) Hülle die Geschwindigkeit drauf die diese vertragen. Welche Geschwindigkeit ihr Brenner unterstützt entnehmen Sie bitte ihrer Hardware Dokumentation.

DVD-Grösse: Hier stellt man die DVD Grösse des Rohlings ein. Zur Auswahl stehen 4,7, 9,4 und 15,9 Gb.

DVD-Speed: Hier stellen Sie die Brenngeschwindigkeit für DVD's ein. Entnehmen Sie dies bitte der Dokumentation zu Ihren DVD Brenner.

Brenner: Hier wird Ihnen eine Auswahl von Brennern angeboten, die in ihrem Rechner vorhanden sind.

Notiz: Es werden nur bis zu 10 Laufwerke unterstützt.

Leselaufwerk: Hier wird Ihnen eine Auswahl von Leselaufwerken angeboten, die in ihrem Rechner vorhanden sind.

Notiz: Es werden nur bis zu 10 Laufwerke unterstützt.

Quell-Treiber: Hier wird Ihnen eine Auswahl von Treibern angeboten, die cdrdao unterstützt. Wählen Sie einen Treiber für Ihr Leselaufwerk aus. Wenn keiner für Ihren Brenner dabei ist, versuchen Sie generic-mmc oder generic-mmc-raw.

Treiber: Hier wird Ihnen eine Auswahl von Treibern angeboten, die cdrdao unterstützt. Wählen Sie einen Treiber für Ihr Schreiblaufwerk aus. Wenn keiner für Ihren Brenner dabei ist, versuchen Sie generic-mmc oder generic-mmc-raw.

CDDB: Hier können Sie ein oder mehrere CD Datenbank Server, durch ein Komma getrennt, eingeben. Format ist:
<server - benutzt standardmaessig das cddbp Protokoll auf Port 888.
<server:<port - benutzt den angegebenen Port fuer das Protokoll cddbp.
<server:<cgi-bin-path> - benutzt standardmaessig das http Protokoll auf Port 80, mit der URL cgi-bin-path>. Standard ist ~/cddb/cddb.cgi.
<server>:<port>:<cgi-bin-path> - benutzt das http Protokoll auf dem angegebenen Port, mit der URL <cgi-bin-path>. Standard ist ~/cddb/cddb.cgi.
<server>:<port>:<cgi-bin-path>:<proxy-server> - benutzt standardmaessig das http Protokoll auf Port 80 und benutzt zur Verbindung den angegebenen Proxy Server.
<server>:<port>:<cgi-bin-path>:<proxy-server>:<proxy-port> - benutzt das http Protokoll auf dem angegebenen Proxy Port zur Verbindung ueber den Proxy Server zur CD-Datenbank.

Tmp Verzeichnis: Wenn die CD/DVD nicht on-the-fly kopiert werden soll benötigen Sie ein temporäres Verzeichnis. Dies gilt auch wenn Sie einen CDDB Server abfragen. In diesem Verzeichnis werden die Dateien der CD/DVD's abgelegt.

CD/DVD auswerfen: Soll die CD/DVD nach dem brennen ausgeworfen werden(yes/no)?

Simulieren: Soll der Brennvorgang nur simuliert werden (ja/nein)?


2 Audio-CD

Nur Audio-Dateien werden zur Auswahl angeboten, die dann anschliessend auf CD gebrannt werden.

Notiz: Zur Zeit gehen nur die Audio Formate mp3, ogg und wav. Für andere Formate sollte man den Menüpunkt Daten-CD benutzen.


3 Audio-DVD

Nur Audio-Dateien werden zur Auswahl angeboten, die dann anschliessend auf DVD gebrannt werden.

Notiz: Zur Zeit gehen nur die Audio Formate mp3, ogg und wav. Für andere Formate sollte man den Menüpunkt Daten-DVD benutzen.


4 CD-kopieren

Hier können Sie eine 1 zu 1 Kopie Ihrer CD brennen. Dabei wird/werden die von Ihnen eingegebenen CD Datenbank Server nach Titel und Interpret abgefragt. Dies kann on-the-fly geschehen oder mit dem Zwischenschritt des Anlegens temporärer Dateien/Daten.


5 DVD-kopieren

Hier können Sie eine 1 zu 1 Kopie Ihrer DVD brennen. Dies kann on-the-fly geschehen oder mit dem Zwischenschritt des Anlegens temporärer Dateien/Daten.


6 Daten-CD

Hier können Sie Verzeichnisse oder einzelne Dateien auf CD brennen. Man kann zur Zeit nur ein Grund-Verzeichnis auswählen und dann die darunterliegenden Verzeichnisse und Dateien.


7 Daten-DVD

Hier können Sie Verzeichnisse oder einzelne Dateien auf DVD brennen. Man kann zur Zeit nur ein Grund-Verzeichnis auswählen und dann die darunterliegenden Verzeichnisse und Dateien.


8 ISO-CD

Nur Iso-Images werden zur Auswahl angeboten, um sie anschliessend auf CD zu brennen. Diese müssen alle im selben Verzeichnis liegen


9 ISO-DVD

Nur Iso-Images werden zur Auswahl angeboten, um sie anschliessend auf DVD zu brennen. Diese müssen alle im selben Verzeichnis liegen


10 Photo-CD

Nur Bilder werden zur Auswahl angeboten, um sie anschliessend auf CD zu brennen. Man kann zur Zeit nur ein Grund-Verzeichnis auswählen und dann die Photo Dateien in diesem und in den darunterliegenden Verzeichnissen.


11 Photo-DVD

Nur Bilder werden zur Auswahl angeboten, um sie anschliessend auf DVD zu brennen. Man kann zur Zeit nur ein Grund-Verzeichnis auswählen und dann die Photo Dateien in diesem und in den darunterliegenden Verzeichnissen.


12 Video-CD

Brennen von Video-CD's. Es werden nur solche Filme zur Auswahl angeboten. Diese müssen alle im selben Verzeichnis liegen


13 Video-DVD

Brennen von einer Video-DVD. Es wird nur ein DVD Film Verzeichnis zur zur Auswahl angeboten.


14 CD-löschen

Hier kann man eine CD+-RW löschen. Dabei wird geprüft ob eine CD im Laufwerk ist.


15 DVD-formatieren

Hier kann man eine DVD+-RW oder DVD-Ram formatieren. Dabei wird geprüft ob eine DVD im Laufwerk ist.
