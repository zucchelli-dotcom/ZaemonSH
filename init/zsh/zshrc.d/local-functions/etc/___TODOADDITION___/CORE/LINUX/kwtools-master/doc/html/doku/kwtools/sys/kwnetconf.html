1.5.9 kwnetconf

kwnetconf ist ein Frontend für verschiedene Scripte die die Konfiguration des Netzwerks betreffen.

Notiz: Alle hier aufgeführten Scripte benötigen root Rechte. benutzt.


kwnetcardconf

 -

Mit kwnetcardconf kann man alle vorhandenen Netzwerkkarten konfigurieren einschliesslich des Standard Gateway. Bei der Konfiguration des WLAN kann man zusätzlich die Parameter der Verbindung einstellen.

pppoeconf

 -

Einwahl per DSL-Modem.

isdnconfig

 -

Einwahl per ISDN.

kwwvdialconf

 -

Einwahl per Analog Modem/Handy.


pppoeconf

Ein Script das einem die Einrichtung des Internet über DSL erleichtert. Sollte eigentlich jeder Distribution beiliegen. Es erstellt die Datei /etc/ppp/peers/dsl-provider mit Ihren Daten. Gegebenfalls muss diese noch angepasst werden. Lesen Sie dazu die Hilfe von pppoeconf.


isdnconfig

Ein Script das einem die Einrichtung des Internet über ISDN erleichtert.
