kwvmstat

kwvmstat gibt Informationen über Prozesse, Speicher, Speicherseiten (paging), Block Input/Output (IO) und der Aktivität des Prozessors aus. Genaueres steht in der Manpage von vmstat(8). Einige Optionen kann man in der Datei ~/.kwtools/kwvmstatrc einstellen.

Notiz: Dieses Script benutzt das Programm vmstat.

1 Konfiguration

Hier kann man einige Optionen einstellen. Siehe auch man kwvmstatrc(5).

Anzahl: Anzahl der Aktualisierungen. 0 geht nicht, nichts ist eine Endlos Anzeige. Diese kann man mit STRG+C beenden.

Verzögerung: Nummer für die Verzögerung der Aktualisierungen (Sekunden). 0 geht nicht.

Speichereinheit: Speichereinheit der Anzeige. k - 1Kb=1000b, K - 1Kb=1024b, m - 1Mb=1000000b, M - 1Mb=1048576b.


2 Optionen

Hier kann man einige Optionen für die Anzeige auswählen.

a - Anzeige von in/aktivem Speicher.

d - Statistik über Festplatten.

D - Anzeige der Festplattentabelle.

f - Nummer der forks seit boot.

m - Anzeige slabinfo.

p - Statistik über Partitionen. Benötigt "root" Rechte.

s - Anzeige von verschiedenen Speicher Statistiken.
