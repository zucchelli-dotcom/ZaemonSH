Funktionen und Nachrichten für den Multimediabereich.

Funktionen: Man muss diese Funktion (multi_conf) nur ausführen, dann kann man alle enthaltenenen Funktionen ausführen, wie z. B. "audio_burn". Wenn nichts zu der Syntax steht, wird die Funktion einfach nur aufgerufen. Manche Funktionen haben auch noch eigene Funktionen die aber nur von diesen benutzt werden und dadurch bisher noch nicht weiter erklärt werden.

audio_burn

Funktion um Audio Dateien zu brennen. Zur Zeit gehen nur die Formate mp3, ogg und wav.
audio_burn [ CD | DVD ]
Es wird dann dementsprechend die Funktion cd_dvd_burn mit der richtigen Option aufgerufen.

cd_device

Funktion zur Auswahl eines CD/DVD Laufwerks. Sind keine vorhanden kommt eine Fehlermeldung.
cd_device
Variable
DEVICE - Variable die das ausgewählte Laufwerk enthält.

cd_driver

Auswahl eines Treibers für Ihre Laufwerke. Zur Zeit wird nur "cdrdao" unterstützt.
cd_driver [ $DRIVER:FLAGS ]
DRIVER - Variable die den vorherigen Treiber enthält.
FLAGS - Variable die die vorherigen Flags für den vorherigen Treiber enthält.
Variable
CD_DRIVER - Variable die den ausgewählte Treiber enthält.

cd_dvd_burn

Dies ist die eigentliche Brennfunktion. Sie liest als erstes die Konfigurationsdatei ~/.kwtools/cd_dvdrc ein, falls sie existiert.
cd_dvd_burn [ --copycd | --datacd | --isocd | --mp3cd | --oggcd | --photocd | --wavcd | --vcd ] oder
cd_dvd_burn [ --copydvd | --datadvd | --isodvd | --mp3dvd | --oggdvd | --photodvd | --wavdvd | --vdvd ].
--copycd, --copydvd - kopiert eine CD oder DVD auf einen neuen Rohling.
--datacd, --datadvd - brennt Daten CD oder DVD.
--isocd, --isodvd - brennt nur ISO Images auf CD bzw DVD.
--mp3cd, --mp3dvd - brennt nur MP3 Dateien auf CD oder DVD.
--oggcd, --oggdvd - brennt nur OGG Dateien auf CD oder DVD.
--photocd, --photodvd - brennt nur Bilder auf CD oder DVD.
--wavcd, --wavdvd - brennt nur Audio CDs oder DVDs.
--vcd - brennt eine Video CD.
--vdvd - brennt eine Video DVD.

cd_dvd_conf

Diese Funktion überprüft, ob alle nötigen Werte zum brennen, nur lesen oder lesen und schreiben eingestellt sind. Wenn nicht wird die dementsprechende Funktion dazu aufgerufen, wie z. B. cd_device.
cd_dvd_conf [ -a | -R | -W ]
-a - Alle Werte überprüfen.
-R - nur die Werte zum lesen einer CD/DVD überprüfen.
-W - nur die Werte zum brennen einer CD/DVD überprüfen.

cd_dvd_config

Die Funktion schreibt die Brenneinstellungen in die Datei ~/.kwtools/cd_dvdrc. Diese Funktion wird nur vom Script kwrecord benutzt.

cd_dvd_speed

Eingabe der Brenngeschwindigkeit (z.B. 0-50). Standart ist 4-fache Geschwindigkeit bei CD-R(W), 1-fache bei DVD-+R(W)/RAM. Es wird dazu die Funktion number_input benutzt.
cd_dvd_speed [ $VAR ]
VAR - Variable die den vorherigen Wert enthält.
Variable
NUMBER - Variable die die eingebene Geschwindigkeit enthält.

cd_size

Auswahl der CD Rohling Grösse die man brennen möchte. Es stehen 650 MB, 680 MB, 700 MB und 800 MB zur Verf&uul;gung.
cd_size
Variable
CD_SIZE - Variable die die ausgewählte CD Grösse enthält.

check_cd_state

check_cd_state überprüft den Status der eingelegten CD, mittels dem Programm "cdrdao".
check_cd_state [ DEVICE ]
DEVICE - CD Blockgerät, Bsp.: /dev/hdc
Variablen
CD_RW - Variable die den Wert enthält ob dies eine wieder beschreibbare CD ist (yes/no).
CD_R_EMPTY - Variable die den Wert yes/no enthält, ob die CD-+R(W) leer ist oder nicht.
CD_CAP - Variable die gesamte Grösse der CD-+R(W) enthält.

check_dvd_state

check_dvd_state überprüft den Status der eingelegten DVD, mittels dem Programm "dvd+rw-mediainfo".
check_dvd_state [ DEVICE ]
DEVICE - DVD Blockgerät, Bsp.: /dev/hdc
Variablen
DVD_RW - Variable die den Wert enthält ob dies eine wieder beschreibbare DVD/DVDRAM ist (yes/no).
DVD_LEGACY - Variable die die tatsächliche Grösse der beschriebenen DVD in Byte enthält.
DVD_LEGACY_BLOCK - Variable die die tatsächliche Grösse der beschriebenen DVD in Blöcken enthält.
DVD_CAP - Variable die gesamte Grösse der DVD-+R(W)/RAM enthält.
DVD_R - Variable die Status der DVD-+R(W)/RAM enthält (complete|blank).

check_exec_dd

Diese Funktion sucht erst nach dem Programm "dd_rescue", ansonsten benutzt sie "dd" zum kopieren einer DVD-+R(W)/RAM in eine Image Datei.
check_exec_dd

dvd_size

Auswahl der DVD Rohling Grösse die man brennen möchte. Es stehen 4,7 GB, 9,4 GB und 15,9 GB zur Verf&uul;gung.
dvd_size
Variable
DVD_SIZE - Variable die die ausgewählte DVD Grösse enthält.

split_size

Auswahl ab welcher Grösse eine Datei gesplittet werden soll. Z.B. kann man dies verwenden damit eine Datei auf den richtigen CD/DVD Rohling passt.
split_size
Variable
SPLIT_SIZE - Variable die die ausgewählte Grösse enthält.

Nachrichten und Fehlermeldungen
change_cd_msg

Aufforderung das Sie jetzt einen neuen leeren Rohling einlegen können, um mit dem Brennprozess fortfahren zu können.

no_cddvd_prog_msg

Info darüber, das Sie erst ein Brennprogramm (CD/DVD) mit kwrecord konfigurieren müssen. Es werden zur Zeit nur cdrdao und growisofs unterstützt.

no_data_medium_msg

Fehlernachricht das Sie erst ein Medium mit Daten einlegen müssen.

no_empty_medium_msg

Fehlernachricht das Sie erst einen leeren Rohling einlegen oder den vorhanden erst loeschen/formatieren müssen.

no_empty_medium1_msg

Fehlernachricht das Sie einen leeren Rohling einlegen oder den vorhanden erst loeschen/formatieren. Frage, ob Sie einen neuen leeren Rohling einlegen oder abbrechen wollen.

over_medium_size_msg

Fehlernachricht wenn die Auswahl der zu brennenden Dateien die Grösse der CD/DVD übersteigt.

over_medium1_size_msg

Fehlernachricht wenn die Auswahl der zu brennenden Dateien die Grösse der CD/DVD übersteigt. Frage, ob Sie einen leeren neuen Rohling einlegen oder abbrechen wollen.

over_tmp_size_msg

Fehlernachricht wenn die Grösse der Auswahl die Grösse des temporären Verzeichnisses übersteigt.
