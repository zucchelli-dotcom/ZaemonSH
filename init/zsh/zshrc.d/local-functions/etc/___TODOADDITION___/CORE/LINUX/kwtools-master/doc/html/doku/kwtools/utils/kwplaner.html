kwplaner

Mit diesem Script können Sie ihre Termine verwalten, Feiertage anzeigen lassen, Kalender drucken etc.. Es vereint die Scripte kwholiday, kwtex-cal und kwtermin.

Notiz: Diese Scripts werden ausschliesslich über kwplaner konfiguriert.


kwholiday

 -

zeigt die Feiertage, entsprechend ihres konfigurierten Landes/Bundeslandes an.

kwtermin

 -

können Sie ihre Termine verwalten. Termine werden per Cron Dämon an Sie und eventuell angegebene User gemailt

kwtex-cal

 -

kann einen Jahreskalender mit Bildern erzeugen und auch ausdrucken.

1 Konfiguration

Hier kann man Land/Bundesland, Mailadresse, Kalenderansicht, den Pfad zu den "calendar" Dateien und den Rechner auf dem kwplaner läuft festlegen. Lesen Sie dazu die Manpage kwplanerrc(5).

Kalenderansicht: Standard Kalenderansicht, wöchentlich, monatlich, ....

Kalender: Dateien in denen nach festen Kalenderdaten gesucht wird. Es können mehrere ausgewählt werden.

Mailadresse: Eingabe der eigenen Mailadresse.

Rechner: Hier kann man den Pechner angeben auf dem kwtermin (localhost oder lokaler Rechnername) läuft.

Notiz: Zur Zeit geht nur der lokale Rechner.

Staat/Bundesland: Hier kann man seinen Staat und das dazugehörige Bundesland auswählen, falls vorhanden. Alle Datensätze die mit dem # Zeichen enden, besitzen eine unvollständige Feiertagsliste. Dies wird für Bundesland spezifische Feiertagsanzeige benötigt.

Verzeichnis: Auswahl des Verzeichnisses in denen nach den Kalender Feiertagen gesucht wird. Standard ist /usr/share/calendar und $HOME/.calendar.
