Hauptfunktion zur Initialisierung der Scripts
script_init

Dies ist die Funktion die jedes Script als erstes einliest. Sie initiiert sozusagen alles was das Script benötigt. Es wird nach dem Programm "dialog" gesucht und deren Version ermittelt, einige Buttons in der eingestellten Sprache belegt und Pfade in denen nach Funktionen oder Programmen gesucht wird eingestellt. Es überprüft die Grösse des Terminal und lädt Funktionen die von den meisten Scripts benutzt werden. Die meisten Funktionen dazu sind in der Funktion ../kwtools/functions/sys/sys_conf enthalten.
scrip_init [-o]
-o - wird nur von Scripts benötigt, die per Cron ausgeführt werden. Es wird kein Terminal benutzt.
