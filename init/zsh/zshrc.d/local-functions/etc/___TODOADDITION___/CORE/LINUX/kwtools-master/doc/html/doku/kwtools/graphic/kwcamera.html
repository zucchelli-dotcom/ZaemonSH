kwcamera

Dieses Script kann ihre Digital Kamera/Camcorder bedienen, soweit es das Programm gphoto(2) >=2.1 unterstützt.

Notiz: Nicht alle USB Geräte lassen sich damit bedienen. Bei vielen neuen USB Geräten braucht man nur noch das richtige Device mounten. gphoto(2) unterstützt serielle und usb Ports.

1 Konfiguration

Hier kann man das Standard Verzeichnis, zum speichern der Bilder, den Port (serial,usb) und das Modell auswählen und abspeichern.

Verzeichnis: Hier wählt man sein Grundverzeichnis aus. Darunter kann man dann später seine Bilder speichern. Ist es nicht vorhanden wird man gefragt, ob es angelegt werden soll.

Port: Hier kann man den Port auswählen an dem die Kamera hängt. Es werden nur vorhandene Ports (dev/ttyS0, ...) angezeigt.

Modell: Hier kann man aus einer Liste, sein Kamera Modell auswählen, falls es von gphoto(2) unterstützt wird.


2 Download

Hier kann man entweder alle Bilder mit einem mal oder per Auswahl von der Kamera in ein Verzeichnis Ihrer Wahl laden. Es gibt auch eine Bildvorschau mittels dem Programm "zgv". Die remote Anzeige funktioniert nicht, die Anzeige wird auf dem entfernten Rechner dargestellt. Ob auch Thumbnails runtergeladen werden sollen wird nachgefragt.


3 löschen

Löschen aller Bilder auf der Kamera.


4 Kamera_Einstellung

Hier kann man seine, von gphoto(2) unterstützten, Kameras, einstellen, wie Uhrzeit, Qualität, etc..
