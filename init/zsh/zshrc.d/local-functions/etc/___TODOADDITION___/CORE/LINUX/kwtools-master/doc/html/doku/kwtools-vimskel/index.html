Beschreibung

Dieses Paket enthält eine Sammlung von Dateien die die Arbeit eines Benutzers, mit dem Editor "vim", leichter machen können.
Dazu gehört das automatische generieren vom Kopf einer neuen Datei/Script, automatische Rechtschreibprüfung (de, en), Kalender, SQL-Unterstützung, .... Es installiert sich in das Verzeichnis /etc/skel.

Beim Erzeugen eines neuen Benutzers wird aus dem Verzeichnis /etc/skel das Verzeichnis "etc" und die dementsprechenden Links in das Heimat-Verzeichnis des Benutzers kopiert.

Notiz: Die Doku ist leider noch "Work in Process", da ich dieses Jahr (2010) kaum Zeit hatte.
