1.5.3 kwcrypt

kwcrypt ist ein Wrapper für kwcryptsetup (cryptetup), kwecryptfs (ecryptfs) und kwlosetup (loo-aes-utils). Es werden nur die Scripte angezeigt die auch installiert sind. Mit diesen Scripte kann man Container (Datei mit Dateisystem), Partitionen (auch Swap Partitionen), Verzeichnisse und Logische Volumes verschlüsseln.

Notiz: Alle Scripts benötigen root Rechte.


kwcryptsetup

 -

Erstellung von verschlüsselten Containern und Block Geräten, wie Festplatten.

kwecryptfs

 -

Datei weise Verschlüssellung von Verzeichnissen.

kwlosetup

 -

Erstellung von verschlüsselten Containern und Block Geräten, wie Festplatten.
