System-Programme

Alle hier vorgestellten Scripte benötigen meist Root Rechte. Mit diesen Scripten kann man weitgehend sein System konfigurieren. Wenn nicht selbsgeschriebene Scripte zum Einsatz kommen wird dieses erwähnt.


kwauth

 -

Mit diesem Script kann man die Methode zur Authentifizierung des Systems einstellen.

kwclock

 -

Zum Einstellen/Anschauen der Uhrzeit und des Datums.

kwcrypt

 -

Ein Wrapper für kwcryptsetup kwecryptfs und die kwlosetup.

kwfuse

 -

Konfigurations Frontend für FUSE (filesystem in userspace).

kwgroup

 -

Anlegen, ändern und löschen von Gruppen.

kwgrub2

 -

Bootloader grub2 installieren, etc. ...

kwlogrotate

 -

Einstellung der Rotation und Archivierung von Logdateien.

kwlvm

 -

Anzeige, Erstellung, Löschen, ..., von PV's, VG's und LV's.

kwnetconf

 -

Konfiguration vorhandener Netzwerkkarten.

kwparted

 -

Ein Frontend für parted, einem Partitionierungstool.

kwpasswd

 -

Mit diesem Script kann man Benutzer Passwörter ändern (NIS und passwd).

kwquota

 -

Ein Frontend für quota, einer Software zur Platzbeschränkung.

kwraid

 -

Erstellung und Kontrolle von Software RAID Arrays.

kwrunlevconf

 -

Zum Einstellen der Runlevel, für die einzelnen Start-Scripts (sysv-rc und file-rc).

kwuser

 -

anlegen, ändern und löschen von Benutzern.
