kwxmms2

kwxmms2 ist ein Script mit welchem man das X-Music-Multiplexing-System 2 konfigurieren und bedienen kann. Wenn Sie einen remote Server (xmms2d) benutzen, sollten Sie darauf achten die selbe Version zu benutzen.

Notiz: Dieses Script benutzt die Programme nyxmms2 und xmms2.


1 Konfiguration

2 Player

3 Playlisten

3 Plugins


1 Konfiguration

Hier können Sie bisher nur eine Einstellung vornehmen.

Serveradresse: Ist der Server online wird die Adresse angezeigt. Diese kann man nicht ändern und dient nur zur Information.

Client: Tragen Sie bitte die Adresse des xmms2d Servers ein. (Bsp.: tcp://host:9667 oder unix:///tmp/xmms-ipc-username). Der IPC Socket "unix" ist nur fuer lokale clients vorgesehen, der Socket "tcp" funktioniert lokal wie remote.

NOTIZ: Es können mehrere Adressen, getrennt durch ein Semikolon angegeben werden.


2 Player

Hier können Sie den Player bedienen.

aktualisieren: Aktualisiert den Kopf der Menübox mit Playlist, Liedname und bisher abgespielter Zeit des Liedes.

Equalizer: Hier können Sie den Equalizer des Players bedienen. Es werden nur Werte zwischen -20 und 20 unterstützt.

Auswahl: Auswahl des abzuspielenden Liedes.

Start: Start des gestoppten, oder ausgewählten Liedes.

Pause: Das momentane Lied pausiert an gestoppter Stelle.

Stop: Stoppt das momentane Lied.

nächstes: Das nächste Lied wird gespielt.

vorheriges: Das vorherige Lied wird gespielt.


3 Playlisten

Hier können Sie die Playlisten bearbeiten.

Ansicht: Ansicht des Inhalts einer ausgewählten Playliste.

Auswahl: Hier können Sie ein Playliste auswählen.

anlegen: Hier können Sie eine Playliste anlegen.

füllen: Hier können Sie Lieder, Playlisten, Audio-CD/DVD Verzeichnisse oder eine URL einer Playliste hinzufügen. Dies hängt von den installierten Plugins ab.

leeren: Damit werden alle Lieder einer ausgewählten Playliste gelöscht.

sortieren: Hiermit können Sie die Lieder einer Playlist sortieren.

mischen: Hiermit können Sie die Lieder einer Playlist mischen.

umbenennen: Hier können Sie eine ausgewählte Playliste umbenennen.

Lieder löschen: Damit können Sie einzelne Lieder einer ausgewählten Playliste löschen.

löschen: Hier können Sie eine oder mehrere ausgewählte Playlisten löschen.
