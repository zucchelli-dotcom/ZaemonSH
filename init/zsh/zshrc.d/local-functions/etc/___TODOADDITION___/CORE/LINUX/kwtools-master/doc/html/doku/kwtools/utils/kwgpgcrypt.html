kwgpgcrypt

Mit diesem Script können Sie ganze Verzeichnisse oder auch nur einzelne Dateien verschlüsseln, signieren, beides oder entschlüsseln. Es besitzt ein Schlüssel-Management so das sie ganz einfach Schlüssel bearbeiten können.

Notiz: Dieses Script benutzt das Programm gnupg.


1 Konfiguration

2 Schlüssel_Management

3 signieren

4 verschlüsseln

5 signieren/verschl.

6 entschlüsseln


1 Konfiguration

Hier kann man den Schlüssel_Server eingeben und die Standard-Schlüssel-ID auswählen, mit der standardmässsig signiert, verschlüsselt, etc. werden soll.

Das Schlüssel_Server Formate sind:
Protokoll://pgpserver:Port. Protokolle können hkp und ldap sein. hkp steht für http oder kompatible Server.
Protokoll:user@pgpserver. Protokoll ist mailto und Mailadresse könnte z. B. pgp-public-keys@keys.pgp.net sein.

Notiz: Die Standard_Schlüssel_ID_lang dient nur zur Anzeige und kann nicht verändert werden.


2 Schlüssel_Management

Hier kann man alle Schlüssel ansehen, Schlüssel durch einen Server aktualisieren/hochladen, Schlüssel für ungültig erklären, exportieren, importieren, signieren und einen eigenen erzeugen.


3 signieren

Signieren von einzelnen Dateien oder ganzen Verzeichnisssen.


4 verschlüsseln

Verschlüsseln von einzelnen Dateien oder ganzen Verzeichnisssen.


5 signieren/verschl.

Signieren und verschlüsseln von einzelnen Dateien oder ganzen Verzeichnisssen.


6 entschlüsseln

Entschlüsseln von einzelnen Dateien oder ganzen Verzeichnisssen.
