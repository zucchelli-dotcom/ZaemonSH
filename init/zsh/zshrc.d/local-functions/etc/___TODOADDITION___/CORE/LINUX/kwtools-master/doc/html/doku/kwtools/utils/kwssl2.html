kwssl2

kwssl2 ist ein simples Script, geschrieben in zsh/dialog, zum verwalten von einer kleinen CA (Certifications Authority). kwssl2 ist ein Frontend für das Programm openssl. Als Vorlage diente das Programm TinyCA2.

Notiz: Dieses Script ist noch Alpha!

1 Fähigkeiten

Es werden folgende Fähigkeiten von kwssl2 unterstützt:

  • unendliche Anzahl an CAs möglich
  • Konfiguration für die Erstellung von CA, Server Zertifikate, Benutzer Zertifikate und Widerrufslisten
  • Erstellung und Verwaltung von SubCAs
  • Erstellung und Widerrufung von x509 - S/MIME Zertifikaten
  • Import von CAs und deren Zertifikate
  • Import und Signierung von PKCS#10 Anforderungen
  • Erstellung von Diffie Hellman Dateien
  • RSA und DSA Schlüssel können erstellt werden
  • CA (Certifications Authority)
    • CAs können nach PEM, DER und TXT exportiert werden
  • Server Zertifikate
    • Zertifikate können nach PEM, DER, TXT und PKCS#12 eportiert werden
    • Zertifikate können zum Beispiel für Apache, Postfix, OpenLDAP, Cyrus, FreeS/WAN, OpenVPN, OpenSWAN, FreeRadius, ..., benutzt werden
  • Benutzer Zertifikate
    • Zertifikate können nach PEM, DER, TXT und PKCS#12 eportiert werden
    • Zertifikate können zum Beispiel für Netscape, Konqueror, Opera, Internet Explorer, Outlook (Express) und FreeS/WAN benutzt werden
  • Widerrufsliste (Certificates Revocation List)
    • CRLs können nach PEM, DER und TXT exportiert werden
