dialog

dialog ist ein Programm mit dem man verschiedene Typen von Dialogboxen, für Shell Scripts, anzeigen kann. Damit können User interaktiv mit Shell Scripts kommunizieren.

Dokumentation

Bedienung:

Die Dialogboxen kann man mit der Tastatur oder falls "gpm" installiert ist mit der Maus bedienen. Bei Verwendung von "gpm" kann man mit der Maus den Menüpunkt anklicken und dann "ENTER" drücken.

Bei Verwendung der Tastatur dienen folgende Tasten zur Bedienung:
Pfeiltasten - hoch, runter, links, rechts, Zahlen hoch/runter zählen.
Tabtaste - zwischen den Fenstern/Auswahl springen
Leertaste - Verfollständigung des Pfades/Dateinamens vom im linken Fenster markierten Verzeichnis.
Enter - Auswahl bestätigen

Im Dokumentationsverzeichnis finden Sie eine Menge Beispiele, für fast alle Dialogboxen. Wenn Sie ein rpm- oder deb Paket installiert haben liegt dies unter /usr/share/doc/dialog/examples
