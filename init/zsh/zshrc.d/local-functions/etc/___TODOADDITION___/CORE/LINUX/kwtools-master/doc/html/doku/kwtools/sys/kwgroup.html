kwgroup

kwgroup legt Gruppen an, löscht und verändert vorhandene Gruppen. Dabei wird geprüft, ob die Id Nummer der Gruppe schon vorhanden ist.

Notiz: Benötigt root Rechte.


1 Konfigurations Optionen


1 Konfigurations Optionen

Gruppe: Wählen Sie einen vorhandene Gruppe aus oder geben einen neuen Gruppen Namen ein.

Gruppen_ID: Gruppen ID (Identifacates Number) eingeben. Es sind nur Integer Zahlen erlaubt und es wird überprüft, ob diese schon vergeben ist.

Passwort: Passwort für die Gruppe eingeben. Es müssen mindestens 6 Zeichen sein.
