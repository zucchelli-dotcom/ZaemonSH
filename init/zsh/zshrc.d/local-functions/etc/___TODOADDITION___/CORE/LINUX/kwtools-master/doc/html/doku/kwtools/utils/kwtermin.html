kwtermin

Hier können Sie ihre Termine verwalten. Termine werden per Cron Dämon an Sie und eventuell angegebene User gemailt. Man kann Termine auch mit bestimmten Regeln versehen. Dazu gehören tägliche, wöchentliche, monatliche und jährliche Wiederholungen, Geburtstage und Ferien versehen. Geburtstage werden dann jährlich wiederholt.

Die Konfiguration erfolgt über das Script kwplaner, welches verschiedene Scripte zusammenfasst.


1 Termine

2 feste_Kalenderdaten


1 Termine

Hier können Sie Ihre Termine verwalten, wie ansehen, eingeben, ändern und löschen. Vergeben Sie dabei immer einen kurzen bündigen Text als Betreff, da dieser von kwtex-cal benutzt wird (Kalender Erstellung).


2 feste_Kalenderdaten

Ansicht der voreingestellten Kalenderdaten (Geschichte, Literatur, etc.)
