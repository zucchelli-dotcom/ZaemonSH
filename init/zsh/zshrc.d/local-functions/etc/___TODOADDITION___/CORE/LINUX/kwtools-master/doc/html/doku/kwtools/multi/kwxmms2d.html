kwxmms2d

kwxmms2d ist ein simples Script mit welchem man den Server des X-Music-Multiplexing-System 2 konfigurieren. starten und stoppen kann.

Notiz: Dieses Script benutzt das Programm xmms2d.


1 Konfiguration

2 Server starten

3 Server stoppen


1 Konfiguration

Hier können Sie bisher nur eine Einstellung vornehmen.

Server: Eingabe der Adresse auf dem der Server lauschen soll. Tragen Sie bitte die Adresse des xmms2d Servers ein. (Bsp.: tcp://host:9667 oder unix:///tmp/xmms-ipc-username). Der IPC Socket "unix" ist nur fuer lokale clients vorgesehen, der Socket "tcp" funktioniert lokal wie remote.


2 Server starten

Lokalen Server starten.


3 Server stoppen

Lokalen Server stoppen.
