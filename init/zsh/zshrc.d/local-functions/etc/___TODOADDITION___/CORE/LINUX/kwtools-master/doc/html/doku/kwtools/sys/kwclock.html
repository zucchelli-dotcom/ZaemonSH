1.5.2 kwclock

Mit diesem Script kann man die Uhrzeit und das Datum des Rechners einstellen. Entweder per Hand oder per Kontakt zu einem Zeitserver.

Notiz: Benoetigt root Rechte. Bei OpenSuSE wird yast benutzt.


1.5.2.0 Konfiguration

Zeitserver: Einstellung von Zeitservern von denen die Uhrzeit geholt werden soll.

Uhrzeit: Anzeige und Einstellung der Uhr. Sie sollten immer auf Sekunde Null die Uhrzeit speichern, da das Programm "date" die Sekunden immer auf Null setzt.

Datum: Anzeige und Einstellung des Datums. Sie sollten immer auf Sekunde Null die Uhrzeit speichern, da das Programm "date" die Sekunden immer auf Null setzt.

NOTIZ: Als Programme werden date, hwclock und ntpdate benutzt.
