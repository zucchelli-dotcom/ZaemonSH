kwrip

Rippt ganze Audio-CD's oder Video-CD's in ein Verzeichniss ihrer Wahl.

Notiz: Dieses Script benutzt die Programme cdparanoia und vcdxrip.

1 Konfiguration

Hier können Sie das Audio- und Video-Verzeichnis angeben, in welchem die Audio und Video Dateien gespeichert werden sollen, das Leselaufwerk auswählen, und die Mount-Verzeichnisse. Die Mount-Verzeichnisse müssen nur angegeben werden, falls die Partition oder das Verzeichnis nicht ständig ins Dateisystem eingebunden ist.

Audio_Optionen: Geben Sie hier die Optionen für das Audio rippen ein. Es wird cdparanoia benutzt (man 1 cdparanoia). Standard ist "-w" (WAV). Geben Sie nicht das Leselaufwerk und die Option "-B" in den "Audio Optionen" mit an. Das Laufwerk wird selbstständig angehängt, solange das Leselaufwerk konfiguriert ist. Die Option "-B" wird an das Laufwerk angehägt, falls nicht gleichzeitig auch encodiert werden soll. Die Option "-B" steht für Batch Mode und speichert die Sounddateien im Format track#.cdda.*. Wobei "#" die jeweilige Tracknummer ist.

Audio_Mountverz.: Mountverzeichnis auswählen. Dieses Verzeichnis muss schon verher existieren und sollte in der Datei /etc/fstab konfiguriert sein.

Audioverz.: Standard Audioverzeichnis auswählen oder erstellen.

CDDB_verwenden: Soll ein CD Datenbank Server abgefragt werden (ja/nein)?

CDDB_Server: Geben Sie hier bitte einen CD Datenbank Server ein (z.B. freedb.freedb.org).

Leselaufwerk: Auswahl eines der vorhandenen CD/DVD Laufwerke.

Proxy: Proxy Einstellungen für die CD Datenbankabfrage. Sie können entweder IP-Adresse:Port oder Rechnernamen:Port eingeben.

Encoder_benutzen: Sollen die Sounddateien encodiert werden (ja/nein)?

Encoder_Programm: Hier können Sie das Programm zur Encodierung auswählen. Zur Zeit werden nur die Programme "lame" und "oggenc" unterstützt.

Lame_Enc_Opt: Geben Sie hier die Optionen für das Programm "lame" ein (man 1 lame). Standard ist -r3mix -b112. Geben Sie nicht die Option "-B" oder "-d" ein. Diese werden automatisch eingefügt. "-B" gleich Batch Mode und "-d" ist das ausgewählte Laufwerk (s. Leselaufwerk).

Ogg_Enc_Opt: Geben Sie hier die Optionen für das Programm "oggenc" ein (man 1 oggenc). Standard ist -b 128. Geben Sie nicht die Option -o (Outputfile) mit ein, diese wird automatisch benutzt.

Video_Mountverz.: Mountverzeichnis auswählen. Dieses Verzeichnis muss schon verher existieren und sollte in der Datei /etc/fstab konfiguriert sein.

Videoverz.: Standard Videoverzeichnis auswählen oder erstellen.


2 Audio

Hier rippt man entweder eine ganze Audio-CD oder eine Auswahl der darauf gespeicherten Sounddateien. Dabei wird in der angegebenen CD Datenbank nach Author und Titel gesucht und der gerippte Titel dementsprechend umbenannt. Wenn es keine Einträge in der CDDB gibt heissen alle Tracks track#.cdda.* (#=Nummer) und es kommt eine Meldung dazu.

NOTIZ: Nicht immer stimmen die Album- und Liedernamen aus der Datenbank mit denen auf der CD überein. Vergleichen Sie dies bitte!


3 Video

Hier rippt man Video-CD's. Dabei wird der Video-CD Name mit dem von Ihnen eingegebenen Dateinamen getauscht.
