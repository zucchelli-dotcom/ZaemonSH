kwmutt

Mit dem Script kwmutt können Sie Ihren Mail Clienten Mutt konfigurieren und verwalten.

1 Konfiguration

Konfiguration der einzelnen Mutt spezifischen Einstellungen. Es werden noch nicht alle Mutt Variablen unterstützt.

Bei IMAP, POP, SMTP Server, sendmail und lokale Mailboxen Einstellungen kann man in der Konfiguration auswählen, ob dies der Standard sein soll. Es wird dann ein Link (default_smtp, default_server) auf die jeweilige Konfigurationsdatei erstellt.


2 erweiterte Konfiguration

Konfiguration der einzelnen Mutt spezifischen Hook- , beziehungsweise Bewertung- und Anzeige- Einstellungen.


3 Mutt starten

Mutt mit einem ausgewählten Postfach starten. Wenn Sie unter X arbeiten wird ein extra "xterm" Fenster, mit Mutt, geöffnet, ansonsten wird die aktuelle Shell benutzt.


4 Adressbuch

Hier kann man das konfigurierte Adressbuch bearbeiten. Zur Auswahl stehen das Programm "abook" und Mutt eigenes Adressbuch. Für Mutt eigenenes Adressbuch muss die Variable alias_file in der Standard Konfiguration mit einem Wert belegt werden. Ist das Programm "abook" installiert wird es bevorzugt. "abook" wurde in Gemeinschaft mit Mutt entwickelt.


5 Drucker

Verwaltung von Muttprint, das Druckprogramm für Mutt. Dazu wird das Programm kwmuttprintcf aufgerufen, falls es installiert ist (Paket kwtools-utils). Für weitere Details siehe kwmuttprintcf(1).


6 Mailinglisten

Verwaltung der Mailinglisten. Zur Zeit geht die Verwaltung nur mit einem Editor oder mit der Funktion editbox. Das Verhalten, was nun benutzt werden soll, kann man mit kwadmin einstellen oder mit einem Editor in der Datei /etc/kwtools/main.cf.



7 Verzeichnisstruktur

Es erstellt die folgende Datei-Struktur:
~/.kwtools/kwmuttrc - lokale Standard Konfigurationsdatei
~/.kwtools/kwmutt/address - Mutt internes Adressbuch.
~/.kwtools/kwmutt/charsethooks - Einstellung von Aliasen zu Zeichensätzen, die mutt vom Namen her nicht kennt.
~/.kwtools/kwmutt/color[s]* - Dateien für die Farbeinstellungen.
~/.kwtools/kwmutt/crypt - Datei für die Einstellung von PGP und S/MIME verschlüsselten Nachrichten.
~/.kwtools/kwmutt/fcchooks - Datei für die Einstellung zum speichern einer Kopie von einer versendeten Nachricht.
~/.kwtools/kwmutt/fccsavehooks - Datei für die Einstellung zum speichern einer Kopie von einer versendeten Nachricht. Dies ist ein aequalent zur gleichzeitigen Ausführung von den Einstellungen in fcchooks und savehooks.
~/.kwtools/kwmutt/folderhooks - Einstellungen für die Mailboxen.
~/.kwtools/kwmutt/format - Einstellungen wie die Anzeige aussehen soll.
~/.kwtools/kwmutt/gpg - Einstellungen für die Verschlüsselung per gnupg .
~/.kwtools/kwmutt/imap - IMAP Einstellungen.
~/.kwtools/kwmutt/[Rechnername|IP].imap - Einstellungen für den IMAP-Server mit der jeweiligen IP oder Rechnernamen.
~/.kwtools/kwmutt/*.lmb - Dateien mit den Einstellungen für lokale Postfächer.
~/.kwtools/kwmutt/mailinglists - Einträge für Mailinglisten.
~/.kwtools/kwmutt/mboxhooks - Einstellungen zum speichern v. Nachrichten einer Mailbox in eine andere.
~/.kwtools/kwmutt/messagehooks - Einstellungen zur Anzeige des Headers von Nachrichten.
~/.kwtools/kwmutt/mkbd - Einstellungen für die Tastenbelegung.
~/.kwtools/kwmutt/pager - Einstellungen für den Pager.
~/.kwtools/kwmutt/pop - POP Einstellungen.
~/.kwtools/kwmutt/[Rechnername|IP].pop - Einstellungen f�r den POP-Server mit der jeweiligen IP oder Rechnernamen.
~/.kwtools/kwmutt/print - Einstellungen für den Drucker.
~/.kwtools/kwmutt/savehooks - Einstellungen zum speichern von Nachrichten.
~/.kwtools/kwmutt/score - Einstellungen für die Anzeige per Punkte-Bewertung.
~/.kwtools/kwmutt/sendhooks - Einstellungen zum versenden von Nachrichten.
~/.kwtools/kwmutt/smime - Einstellungen für S/MIME.
~/.kwtools/kwmutt/*.smtp - Dateien mit den Einstellungen zum versenden von mails mit dem sendmail Kommando.
~/.kwtools/kwmutt/[Rechnername|IP].smtp - Einstellungen für den SMTP-Server mit der jeweiligen IP oder Rechnernamen.
~/.kwtools/kwmutt/signature - Datei die die Standard Signatur für die Mail enthält.
~/.kwtools/kwmutt/smime - Einstellungen für die Verschlüsselung per S/MIME.
~/.kwtools/kwmutt/sort - Sortierung der Baumansicht.
~/.kwtools/kwmutt/ssl - Einstellungen für die SSL/TLS Verbindung.
~/.kwtools/kwmutt/unignore_headers - Einstellungen für die Anzeige vom Header.
~/.kwtools/kwmutt/urlview - Einstellungen für die Anzeige von Internet Seiten.
