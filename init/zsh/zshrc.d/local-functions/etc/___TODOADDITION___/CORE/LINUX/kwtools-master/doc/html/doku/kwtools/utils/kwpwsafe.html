kwpwsafe

Mit kwpwsafe können Sie Counterpane "Passwordsafe" Benutzername/Passwort Datenbank bearbeiten, wie Datenbanken erstellen, löschen, Datenbankeinträge ansehen, bearbeiten, erstellen und löschen. Siehe Links.

1 Konfiguration

Standard Datenbank: Auswahl einer Datenbank die standardmässig geöffnet werden soll.

Passwortlänge: Einstellen der Passwortlänge die beim generieren eines Passwortes benutzt werden soll.


2 Datenbank öffnen

Hier können Sie eine Datenbank zur Bearbeitung öffnen. Die konfigurierte Standard Datenbank ist bereits markiert.

Gruppe: Auswahl oder Eingabe einer Gruppe zu der der Eintrag gehören soll. Dies dient zur besseren Übersicht, ist aber optional.

Namen: Name des neuen Eintrags.
Zum Beispiel:
Gruppe=haushalt und Name=test -> haushalt.test
Name=test -> test

Benutzernamen: Geben Sie hier Ihren Benutzernamen ein. Das ist der Name zu dem das Passwort gehören soll. Standard ist Ihr Loginname.

Passwort: Eingabe eines Passworts für den Benutzernamen.

Passwort generieren: Hier können Sie ein Passwort generieren. Dabei wird die voreingestellte Passwortlänge benutzt. Ist diese leer beträgt die Passwortlänge 8 Zeichen.

Notiz: Eingabe einer Notiz. Dies geschieht entweder über die dialogbox "editbox" oder über einen Editor Ihrer Wahl. Es kommt drauf an was sie in der Datei /etc/kwtools/main.cf eingestellt haben (man 5 main.cf, Option EDITOR).


3 Datenbanken verwalten

Datenbank erstellen: Eine neue Datenbank erstellen. Dazu müssen Sie einen Datenbanknamen und ein Passwort vergeben. Der Dateiname sollte ohne Dateiendung (*.dat) eingegeben werden. Diese wird automatisch angehängt.

Datenbank exportieren: Eine Datenbank exportieren. Die entstehende Datei besitzt den Namen der Datenbank mit dem Suffix ".txt".

Vorsicht: Diese Datenbank wird im Klartext gespeichert.

Datenbank importieren: Eine Datenbank in eine andere Datenbank importieren. Dabei müssen Sie für beide Datenbanken das richtige Passwort eingeben. Zuerst wird die Datenbank ausgewählt in die die andere Datenbank importiert werden soll und dann die zu importierende Datenbank. Bei der Passwortabfrage wird oben die Datenbank angezeigt in die importiert werden soll und darunter die Passworteingabe. Unter der 1. Passworteingabe steht der Name der zu importierenden Datenbank und darunter die Passworteingabe dafür.

Passwort ändern: Passwort einer Datenbank ändern. Dabei müssen Sie erst das alte Passwort eingeben und dann das neue Passwort 2 mal in die Felder darunter.

Datenbank löschen: Ein oder mehrere Datenbanken löschen.
